# 画面遷移図

## 1. 全体遷移（共通）

```text
[C-01 ログイン]
        ↓
[C-02 ダッシュボード]
        ↓（ロール分岐）
 ┌───────────────┬─────────────────┬─────────────────┐
 │ 生徒          │ 監督・コーチ  　　│ 管理者　　　　　 │
 │ Student       │ Coach           │ Admin           │
 └───────────────┴─────────────────┴─────────────────┘
```

## 2. 生徒向け画面遷移

```text
[C-02 ダッシュボード]
        │
        ├─→ [S-01 欠席申請一覧]
        │        │
        │        ├─→ [S-02 欠席申請作成／編集]
        │        │        ├─（下書き保存）→ [S-01]
        │        │        └─（提出）→ [S-03 欠席申請詳細]
        │        │
        │        └─→ [S-03 欠席申請詳細]
        │                 └─（SUBMITTED時 取消）→ [S-01]
        │
        ├─→ [S-04 試合結果一覧]
        │        │
        │        ├─→ [S-05 試合結果報告作成]
        │        │        ├─（下書き保存）→ [S-04]
        │        │        └─（提出）→ [S-06 試合結果詳細]
        │        │
        │        └─→ [S-06 試合結果詳細]
        │
        └─→ [S-07 練習スケジュール閲覧]
```

## 3. 監督・コーチ向け画面遷移
```text        
[C-02 ダッシュボード]
        │
        ├─→ [CCH-01 承認待ち一覧]
        │        │
        │        └─→ [CCH-02 申請／報告詳細]
        │                 ├─（承認）→ [CCH-01]
        │                 ├─（差し戻し）→ [CCH-01]
        │                 └─（却下）→ [CCH-01]
        │
        └─→ [CCH-03 練習スケジュール一覧]
                 │
                 └─→ [CCH-04 練習スケジュール登録／編集]
                          ├─（登録／編集）→ [CCH-03]
                          └─（削除）→ [CCH-03]
```
